# cz-example
cz工具集示例

## Commitizen

全局安装cz

``` javascript
npm install -g commitizen
```

### cz-conventional-changelog

安装符合Angular规范提交说明的cz适配器

``` javascript
// npm
commitizen init cz-conventional-changelog --save --save-exact
// yarn
commitizen init cz-conventional-changelog --yarn --dev --exact
```
